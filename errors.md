# Error Codes

## 400 Bad Request
Occurs when required fields are missing or the JSON body is invalid.

Example:
{
  "error": "Invalid input",
  "details": "Title is required."
}

## 401 Unauthorized
Your API key is missing or invalid.

## 404 Not Found
The requested resource could not be located.

## 500 Server Error
An unexpected error occurred on the server.
